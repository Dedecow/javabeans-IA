# javabeans-i

MVP inicial do simulador JavaBeans com IA (futuro Azure OpenAI).

## Endpoints atuais
- GET /game/health       → "Game is running!" ou similar
- GET /game/cenario      → "Primeiro cenário de teste"

## Como rodar
mvn spring-boot:run

## Tecnologias
- Spring Boot 3.3.4
- Java 21
- Maven
